### 파이썬 챗봇

---

파이썬은 

| 저장 | 조건 | 반복 |
| ---- | ---- | ---- |

으로 이루어짐

---

1. 저장

   ```python
   a = 30
   b = '안녕'
   ```

   * `=`은 변수에 값을 저장해주는 기호이다.

   * `==`은 동일하다는 의미이다.

     * ex)

     * ```python
       a = 3
       a == 3 => TRUE
       ```

   ---

   2. 조건

      **Boolean 연산자**

      | TRUE | FALSE |
      | ---- | ----- |
      | 1    | 0     |

      `and / &` 는 하나만 `F`여도 결과값이 `F`이다.

      `or / |`은 하나만 `T`여도 결과값이 `T`이다.

   ---

     3. 반복

        * **for in 반복**

          * ```python
            #10번 반복한다
            for i in range(0,11):
            	print(i)
                
            ```

---

### python 여러가지 print 입력방법

**1. Fstring **

```python
a = '학교'
b = '종이 땡땡땡'
print(f'{a} {b}')
#결과물 = 학교 종이 땡땡땡
```

**2. %d **

```python
a = '학교'
b = '종이 땡땡땡'
print("%s %s" % (a,b))
#결과물 = 학교 종이 땡땡땡
```

**3. `.format`**

```python
a = '학교'
b = '종이 땡땡땡'
print("{}{}".format(a,b))
#결과물 = 학교 종이 땡땡땡
```
